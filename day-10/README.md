#### Function with Outputs
```python
def my_function():
    result = 3*2
    return result

# WHEN YOU CALL THE FUNCTION YOU CAN STORE THE RESULT VARIABLE TO ANOTHER VARIABLE
a = my_function()
# THE VALUE OF 'A' WILL BE 6
```
